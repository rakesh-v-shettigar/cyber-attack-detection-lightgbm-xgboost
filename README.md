# Cyber Attack Detection Using Machine Learning

This research project focuses on detecting cyber attacks using machine learning models. The notebook includes implementation of baseline models along with advanced gradient boosting algorithms such as LightGBM and XGBoost.

A hybrid model is also proposed by combining these models to improve the overall performance of cyber attack classification.

# Dataset

1.The dataset used in this project is generated from **Metasploitable 2**, a vulnerable environment used for cybersecurity research and attack simulations.

2.The dataset is divided into two phases for analysis and model evaluation. The dataset files will also be uploaded in this repository.

# Models Used

1.Base Machine Learning Models - SVM, RF , etc

2.LightGBM

3.XGBoost

4.Proposed Hybrid Model of LightGBM & XGBoost on a Meta Learner
